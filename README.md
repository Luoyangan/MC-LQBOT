## MC-LQBOT QQ机器人
**MC-LQBOT** 是一个基于LQBOT框架开发的QQ机器人，对接了MC服务器的**WebItem**插件
- **WebItem**插件可以在MC服务器中开发HTTP/HTTPS&WS/WSS协议的操作接口 [WebItem](https://www.minebbs.com/resources/webitem-web.14228/)

### 功能
- 白名单相关功能
- 在线玩家查询功能
- 银行功能，银行余额可提现至MC服务器
- 签到奖励功能
- 购物功能
- 飞行功能
- 机器人菜单功能

## 技术栈
- 语言：Go
- 框架：LQBOT-0.5.15 (git eba2a31)
